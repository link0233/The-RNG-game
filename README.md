# The-RNG-game
---------------
##### 執行 theRNGgame.exe即可遊玩
##### 執行 seeyouritems.exe 即可看你的物品存檔
##### 更興(0.3後)把saves資料夾內的東西更換成就版本的，基本上都是互通的
